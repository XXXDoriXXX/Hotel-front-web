## Description

**Hotel-front-web** is the web-based admin panel for hotel owners, designed as part of the Hotel management ecosystem. Built with JavaScript, this application provides a powerful, intuitive interface for managing hotels, rooms, reservations, analytics, and more ‚Äî all from your browser.

This admin panel enables hotel owners and administrators to efficiently oversee hotel operations, monitor key metrics, manage bookings, and interact with guests.

## Getting Started

### Prerequisites

- Node.js (v14+ recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/XXXDoriXXX/Hotel-front-web.git
    cd Hotel-front-web
    ```

2. **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3. **Configure environment variables:**  
   Create a `.env` file (if required) and set API endpoints and keys.

4. **Run the development server:**
    ```bash
    npm start
    # or
    yarn start
    ```

5. **Open your browser and navigate to**  
    ```
    http://localhost:3000
    ```
